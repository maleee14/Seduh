## Installation
Clone Projek

```bash
  git clone https://github.com/maleee14/Seduh.git
```

Copy .env.example menjadi .env

```bash
    composer install
```

```bash
    php artisan key:generate
```

```bash
    php artisan storage:link
```

```bash
    php artisan migrate:refresh --seed
```
